## ?? Common Issues & Quick Fixes

| Issue | Quick Fix |
|-------|-----------|
| Backend won't start | Check `.env` file has all 4 values |
| No Google Sheets access | Share sheet with service account email |
| Email not sending | Verify App Password (remove spaces) |
| Frontend blank | Make sure backend is running first |
| Port 3001 in use | Change PORT in `.env` to 3002 |
| Suggestions not showing | Add data to `QnA_Database` sheet |
| Lead form not appearing | Use booking keywords: "book", "reserve" |
